Google Sitemap Module
Author: Matthew Loar <matthew at loar dot name>
Written as part of the Google Summer of Code, 2005

DESCRIPTION
-----------
The Google Sitemap Module creates an XML sitemap at q=gsitemap that conforms to the Google specification.  It allows many options for customization of the data reported in the sitemap.

DEPENDENCIES
------------
None.

INSTALLATION
------------
See INSTALL.txt in this directory.

MORE INFORMATION
----------------
See http://www.google.com/webmasters/sitemap
