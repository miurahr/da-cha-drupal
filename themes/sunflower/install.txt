Readme
------

Found on OSWD.com  (Open Source Web Design).  Listed as sunflower and 
created by 'artwiz'.  I've modified it to my tastes and drupalized it.
Suggestions are welcome.

Send comments to Joe@familytimes.com

Requirements
------------

Drupal 4.5

Installation
------------

Place sunflower.theme and sunflower.css in themes/sunflower.

Log on as an administrator and activate the theme from administer->themes
